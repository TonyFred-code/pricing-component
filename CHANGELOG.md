# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-04-16

### Added

- Added `Toggle` component to handle toggling between pricing terms
- Created `App` component to store pricing component
- Created `Link` component to handle empty links
- Added custom themes, utility classes, and component style rules
- Added linter and formatter (`eslint` and `prettier`)
- Added `husky` and `lintstaged` to lint and format every git commit
- Added `tailwindcss` to style project
- Added `prop-types` package to ensure props validation
